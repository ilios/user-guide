# Courses and Sessions

Clicking on the Courses and Sessions menu item pulls up the screen for maintaining both Courses and Sessions in Ilios. The interface allows for a great deal of information to be displayed and editable at the top level. Please refer to the following sections for more information. Courses are displayed initially. You can drill down into Sessions and related Offerings and Independent Learning Modules, and can update Publishing status of both Courses and Sessions.
