This has to do with learning materials and accessibility. 

## Accessibility Required

This gets set here. This requires that a user uploding a new learning material to Ilios will need to declare that the learning material (file, link, or other) is certified to be accessibile to those who may have individual requirements for full access to the learning material itself.

## Accessibility Requirements Link

A field is provided for a link to be added to provide accessibility information (optional).
